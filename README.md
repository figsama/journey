# 缅甸行程安排

这是一个使用 GitHub Pages 发布的缅甸旅行行程可视化网站。

## 🌐 访问网站

网站地址：`https://YOUR_USERNAME.github.io/journey/`

（将 YOUR_USERNAME 替换为你的 GitHub 用户名）

## 📝 使用说明

1. 首次使用需要：
   - 在 GitHub 仓库设置中启用 Pages
   - 选择 Source: GitHub Actions
   - 推送代码后会自动部署

2. 查看行程：
   - 打开网站即可查看交互式地图和详细行程安排
   - 包含每日行程、预算、关键地点等信息

## 🔧 本地开发

直接在浏览器中打开 `index.html` 文件即可预览。
